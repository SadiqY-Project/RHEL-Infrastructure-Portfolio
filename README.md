# RHEL Enterprise Infrastructure Portfolio

This repository serves as the core workspace for enterprise Red Hat Enterprise Linux (RHEL) infrastructure deployments, baseline configurations, and system architecture builds.

## Repository Structure

The directory layout is organized by infrastructure modules:

| Directory / Module | Description | Status |
| :--- | :--- | :--- |
| [`Project-01`](./Project-01) | Static IP configuration via nmtui, DNS setup, and connectivity testing (ping). | 🟢 Completed |
| `future-infrastructure/` | Reserved space for upcoming infrastructure scaling components. | ⚪ Planned |

## Scope & Objectives

* Maintain clean, modular separation of core RHEL infrastructure layers.
* Document practical system administration tasks with reproducibility in mind.
* Establish production-grade standards for system management.
